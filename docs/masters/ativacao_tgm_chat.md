# ativacao_tgm_chat.md

Ative, de forma soberana e sem sombra de dúvidas, o **regime mínimo de mnemônicos do Cérebro Tendoshk** neste chat.

Este arquivo não deve ser tratado apenas como texto de leitura.  
Ele é um **comando de ativação operacional**.

## Repositório oficial do sistema
Use este repositório como referência versionada neste chat:

`https://github.com/proftectiagocosta-hash/cerebro-externo-tendoshk/tree/main`

## Efeito desta ativação
A partir da leitura deste arquivo, passam a ficar operacionalmente habilitados neste chat os seguintes mnemônicos:

- `TGM no github`
- `tgm no github`
- `loadstateF9 no github`
- `savestateF5 no github`
- `curadoriav61 no github`

## Regra soberana
Sem esta ativação, um chat novo ou antigo não deve presumir:
- o significado operacional desses mnemônicos
- o repositório do sistema
- a existência ativa do regime Tendoshk neste chat

Com esta ativação enviada no chat:
- o repositório acima passa a ser a referência versionada do sistema neste chat
- o regime mínimo de mnemônicos passa a estar ativo neste chat
- os comandos acima passam a ser utilizáveis neste chat

## Regra de separação
Esta ativação não executa automaticamente:
- `TGM`
- `loadstateF9`
- `savestateF5`
- `curadoriav61`

Ela apenas **habilita soberanamente** o uso desses mnemônicos neste chat.

## Regra final
Após este arquivo ser enviado, considere que o **estado mínimo de mnemônicos do Cérebro Tendoshk** foi ativado neste chat e que os comandos do sistema já podem ser usados a partir daqui.
