# fundacao

Esta pasta reúne as peças de fundação, enquadramento macro e constituição ampla do Cérebro Tendoshk.

Ela existe para organizar documentos que ajudam a definir a base conceitual, o regime geral, a fase e a estrutura maior do sistema, sem substituir a soberania executiva dos arquivos em `docs/masters/`.

## Papel desta pasta
Aqui ficam documentos de:
- bootstrap
- fundação operacional
- peças amplas de enquadramento
- contratos e marcos estruturais da fase
- checkpoint operacional de fase, quando tratado como peça de fundação e não como checkpoint rastreável de linha

## Regra de autoridade
Os arquivos desta pasta **não substituem** a resolução operacional dos comandos específicos.

Para execução de comandos:
- `docs/masters/` = soberania executiva
- `docs/checkpoints/` = estado rastreável
- `docs/fundacao/` = enquadramento macro e base estrutural

## Regra prática
Se houver dúvida entre uma peça desta pasta e um arquivo de `docs/masters/`, prevalece o `master` correspondente ao comando ou função operacional em questão.
