# historico

Esta pasta reúne peças históricas, versões intermediárias, documentos supersedidos, transições arquiteturais e materiais que preservam a evolução do Cérebro Tendoshk.

Ela existe para evitar perda de trilha histórica sem deixar documentos antigos concorrerem em soberania com as peças executivas atuais.

## Papel desta pasta
Aqui ficam documentos de:
- versões antigas de prompts mestres
- versões antigas de protocolos
- peças de transição
- documentos relevantes para auditoria da evolução
- materiais que seguem úteis como referência histórica, mas não como autoridade executiva principal

## Regra de autoridade
Os arquivos desta pasta **não devem ser tratados como soberania executiva atual**, salvo indicação explícita e posterior do usuário.

A execução operacional vigente deve priorizar:
- `docs/masters/`
- `docs/checkpoints/`

## Regra prática
Histórico preserva valor.
Histórico não deve disputar autoridade com a camada executiva vigente.
